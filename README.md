# 📱 AI Parental Control System

A self-hosted parental monitoring and control system built with **Python, FastAPI, and Streamlit**.

This project allows parents to monitor device activity, track phone location, enforce bedtime internet restrictions, and remotely control a child's phone.

⚠️ This project is designed for **parents monitoring their own child's device with consent**.

---

# 🚀 Features

### 📍 Location Tracking
Track the latest GPS location of the device and display it on the dashboard map.

### 📱 App Usage Monitoring
Logs apps used on the phone and shows top apps used.

### 💬 Message Monitoring
Capture message previews from notifications.

Detects risky keywords such as:

- drugs
- violence
- explicit content

### 🚫 Block Specific Apps
Automatically removes apps like:

- TikTok
- Snapchat
- Telegram
- Instagram

### 🔒 Remote Phone Control
From the dashboard you can:

- Lock the phone
- Disable internet
- Enable internet

### 🌙 Bedtime Mode
Automatically disables internet during sleeping hours.
